### Siyukio Starter

**Siyukio Starter** is a frontend starter template based on secondary development of the open-source project **Tencent TDesign**.

This project includes and preserves open-source code from TDesign.  
Original repository:  
[tdesign-vue-next-starter](https://github.com/TDesign/tdesign-vue-next-starter)

> ⚠️ This project is **not officially affiliated** with Tencent or TDesign.  
> It is independently built based on publicly available MIT-licensed TDesign code.

### Components

- code-input: Supports displaying JSON and Markdown using CodeMirror.

### Features

- Support for **MCP WebSocket transport**

### Development

```bash
# Install dependencies
pnpm install

# Start development server
npm run dev
```

### Test

```bash
# Start test
npm run test
```

### Debugging in VSCode

You can debug the project using VSCode with a custom [launch.json](./.vscode/launch.json).

### Build

```bash
# Build for production
npm run build

# Build for test environment
npm run build:test
```
