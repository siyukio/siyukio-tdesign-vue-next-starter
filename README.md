# Siyukio TDesign Starter

This is a customized version of [tdesign-vue-next-starter](https://github.com/TDesign/tdesign-vue-next-starter).

It serves as the internal base template for Siyukio projects, extending TDesign’s Vue 3 starter.

## Modifications include

1. Added MCP WebSocket implementation

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development
pnpm dev
```
